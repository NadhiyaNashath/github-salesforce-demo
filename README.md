## Purpose
This is a sandbox environment to practice building and deploying prebuilt 
integrations using the WSO2 Integrator BI low-code editor and Devant platform.

## Current Practice Integration
**GitHub Issue to Salesforce Case**
Automatically creates a Salesforce Case when a GitHub issue is labeled 
with a specific trigger label.

## Notes
- This is a practice repo and not for production use
- The actual integration will be submitted to wso2/integration-samples
